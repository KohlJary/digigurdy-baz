Detailed Description

These are configuration variables that only apply to gear-motor-crank models. USE_GEAR_CRANK must be enabled for these to have effect.

◆ SPIN_SAMPLES

const int SPIN_SAMPLES = 700

The number of voltage samples per update.

This is used to slow down how fast the code runs and avoid timing issues. Choose based on your Teensy unit:

  • Teensy3.5 @ 120MHz = 700 <– full 3.5 speed
  • Teensy3.5 @ 72Mhz = 240 <– this seems to work fine...
  • Teensy3.5 @ 48Mhz = 100 <– more like 160ms... this is so-so-stable.
  • Teensy4.1 @ 600MHz ~ 3850 <– full 4.1 speed... seems unnecessary
  • Teensy4.1 @ 150Mhz ~ 1150 <– seems fine
